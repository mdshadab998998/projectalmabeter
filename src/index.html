Get Youtube Subscribers

Get Youtube subscribe is a simple Api application that makes use of nodejs and mongoDB as well as Nosql database. The goal of this project is to create an api for there are three routes in it.

You can try entering each route using the routes given below

/subscribers : Provides an array of all subscribers details in the database.
/subscribers/namres : Provides an array of all subscribers with only two feilds, their names and subscribed channel.
/subscribers/:id : Provides the details of subscriber with the given ID.
